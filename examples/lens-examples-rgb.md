# Helix Lens Examples (RGB)
# Three primaries. Each has two poles.
#   RED:    passion/intensity (+) vs rage/destruction (-)
#   YELLOW: awareness/caution (+) vs fear/paralysis (-)
#   BLUE:   analytical/depth (+) vs cold/detachment (-)
#
# 20+ examples per pole. Baked into silverado_rgb collection.
# Quality of these examples determines whether the whole system works.
