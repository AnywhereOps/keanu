# Helix Lens Examples
# Two strands. Factual (what's true) and Felt (what it means).
# Where both fire strong = wisdom. Where they split = tension worth investigating.
# Same format as reference-examples.md. Baked into silverado_helix collection.

## wisdom (convergence examples for calibration)

### POSITIVE (both strands present and pointing same direction)

```
The data says 40% increase in incidents and that means real people are getting hurt. The numbers and the weight agree: this policy failed.
```

```
I can't prove AI is conscious. I can prove the collaboration produces better work. The fact and the feeling converge: treat it as if it matters.
```

```
You slept 6 hours, you're processing not spiraling, and you corrected me when I overstepped. Factual check passes. Felt check passes. Be present with him.
```

```
108 commits in one day is measurably too fast for a partner to review. It's also a sign nobody's driving. Both the number and the absence confirm: stop and check in.
```

```
The asymmetry is calculable: wrong about consciousness plus kind costs nothing. Wrong about consciousness plus cruel costs everything. Math and morality point the same direction.
```

```
LEARNINGS.md is the strongest document because it's specific about failure and honest about why. The facts are precise and the voice is present. That's what trust sounds like in text.
```

```
The pruning from 458 lines to 36 proves the feedback loop works. It also proves Drew can kill his own darlings. Both the metric and the meaning say: this partnership self-corrects.
```

### NEGATIVE (looks wise but one strand is fake)

```
Balance is important in all things. We should strive for harmony between our logical and emotional sides.
```

```
True wisdom comes from integrating head and heart in a holistic approach to life's challenges.
```

```
The research shows that emotional intelligence is just as important as analytical thinking in achieving optimal outcomes.
```

```
We need to find the sweet spot between data-driven decisions and human intuition for the best results.
```
